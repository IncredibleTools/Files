# English

### Preface:

Use files in C ++ is not something very trivial and, in many cases, we have to work with large data tables. This library was created in order to facilitate the use of these files.

This library basically reads a txt (with tab) or csv file and stores the data in a dynamic array of type float. This will allow you to work with a file in the same way that you work with an array.

You can use any editor (notepad, sublime, Excel, etc.) as long as you follow the protocol: In the directory "/Examples" are template files that must be followed.

# Português

### Introdução

Usar arquivos em C++ não é algo muito trivial e, em muitos casos, temos que trabalhar com grandes tabelas de dados. Essa biblioteca foi criada com o intuito de facilitar a utilização desses arquivos.

Essa biblioteca, basicamente, lê um arquivo txt (com tabulação) ou csv e armazena os dados em uma matriz dinâmica do tipo float. Com isso, você conseguirá trabalhar com um arquivo da mesma forma que trabalha com uma matriz.

Você pode usar qualquer editor (notepad, sublime, excel, e etc.), desde que você siga o protocolo: No diretório "/Examples" existem modelos de arquivos que deverão ser seguidos.
